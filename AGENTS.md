# 康熙來了問答大賽網站

## 專案概述

## 技術棧

- CSS3
- JavaScript (ES6+)
- Bootstrap 5.3.0 (CDN)
- SweetAlert2 (CDN)

## 開發指南
1. 計時系統實作建議：
   ```javascript
   // 計時器核心功能
   const timer = {
     time: 30,
     interval: null,
     start() {
       this.interval = setInterval(() => {
         this.time--;
         if (this.time <= 0) {
           this.stop();
           // 時間到，跳下一題
         }
       }, 1000);
     },
     stop() {
       clearInterval(this.interval);
     }
   };
   ```

2. HTML 新增計時器元素：
   ```html
   <div class="timer-container">
     <div class="timer-circle">
       <span class="time-left">30</span>
     </div>
   </div>
   ```

3. CSS 計時器樣式：
   ```css
   .timer-circle {
     width: 60px;
     height: 60px;
     border-radius: 50%;
     border: 3px solid #3498db;
     display: flex;
     align-items: center;
     justify-content: center;
   }
   
   .time-warning {
     border-color: #e74c3c;
     animation: pulse 1s infinite;
   }
   ```
