# 康熙來了問答大賽網站

## 專案概述
這是一個純前端的康熙來了節目主題問答遊戲網站，使用 HTML、CSS 和 JavaScript 開發，採用 CDN 方式引入外部套件。

## 技術棧
- HTML5
- CSS3
- JavaScript (ES6+)
- Bootstrap 5.3.0 (CDN)
- SweetAlert2 (CDN)

## 專案目標
該專案旨在建立一個互動性強且具有教育意義的康熙來了問答遊戲網站，讓使用者能透過遊戲回顧節目經典內容，並享受挑戰答題的樂趣。

## 開發指南
1. 計時系統實作建議：
   ```javascript
   // 計時器核心功能
   const timer = {
     time: 30,
     interval: null,
     start() {
       this.interval = setInterval(() => {
         this.time--;
         if (this.time <= 0) {
           this.stop();
           // 時間到，跳下一題
         }
       }, 1000);
     },
     stop() {
       clearInterval(this.interval);
     }
   };
   ```

2. HTML 新增計時器元素：
   ```html
   <div class="timer-container">
     <div class="timer-circle">
       <span class="time-left">30</span>
     </div>
   </div>
   ```

3. CSS 計時器樣式：
   ```css
   .timer-circle {
     width: 60px;
     height: 60px;
     border-radius: 50%;
     border: 3px solid #3498db;
     display: flex;
     align-items: center;
     justify-content: center;
   }
   
   .time-warning {
     border-color: #e74c3c;
     animation: pulse 1s infinite;
   }
   ```
